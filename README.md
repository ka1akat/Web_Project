# Web_Project

KBTU Web Development Project using **Angular** and **Django**.

## Project Description

Web_Project is a full-stack personal finance management web application 
developed as part of the Web Development course at KBTU 
(Kazakh-British Technical University). The application allows users to 
track their daily expenses, manage budgets, and visualize their spending 
habits through an intuitive dashboard interface.

## Key Features

- **User Authentication** — Secure registration and login system with profile management
- **Dashboard Overview** — Personalized dashboard displaying card balance, total spent amount, and expense count
- **Expense Tracking** — Add, view, and manage personal expenses with detailed categorization
- **Budget Management** — Set and monitor monthly budgets with warning notifications
- **Category Breakdown** — Visual pie chart showing spending distribution across categories
- **Daily Expenses Chart** — Timeline visualization of daily spending patterns
- **Card Integration** — Display of linked payment card (VISA) with balance and validity info
- **Motivational Quotes** — Inspirational financial quotes on the dashboard
